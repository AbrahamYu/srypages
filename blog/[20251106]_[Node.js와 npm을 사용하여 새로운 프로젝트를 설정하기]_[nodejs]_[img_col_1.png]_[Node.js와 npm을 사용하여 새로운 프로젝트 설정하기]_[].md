
# Node.js 프로젝트 시작하기: 초보자를 위한 완벽 가이드

Node.js는 확장성 있는 네트워크 애플리케이션을 만들 수 있게 해주는 강력한 JavaScript 런타임입니다. 이 가이드에서는 Node.js와 npm(노드 패키지 매니저)을 사용하여 새로운 프로젝트를 설정하고, 간단한 웹 서버를 만들어보는 과정을 단계별로 안내합니다.

## 사전 준비

프로젝트를 시작하기 전에, 여러분의 컴퓨터에 Node.js와 npm이 설치되어 있어야 합니다. 설치되어 있지 않다면 아래 공식 사이트에서 다운로드하여 설치해주세요. LTS(Long Term Support) 버전을 설치하는 것을 권장합니다.

- [Node.js 공식 다운로드 사이트](https://nodejs.org/)

설치가 완료되었다면, 터미널(Windows의 경우 Command Prompt 또는 PowerShell)을 열고 아래 명령어를 실행하여 설치를 확인합니다.

```bash
node -v
npm -v
```

각각 버전 번호가 출력된다면 성공적으로 설치된 것입니다.

## 1단계: 프로젝트 폴더 생성 및 이동

먼저 프로젝트를 저장할 새로운 폴더(디렉토리)를 만들고, 해당 폴더로 이동합니다.

```bash
mkdir my-node-project
cd my-node-project
```

이제 `my-node-project`라는 이름의 폴더 안에서 모든 작업을 수행하게 됩니다.

## 2단계: npm으로 프로젝트 초기화

새로운 Node.js 프로젝트를 시작할 때 가장 먼저 해야 할 일은 `package.json` 파일을 생성하는 것입니다. 이 파일은 프로젝트에 대한 정보(이름, 버전 등)와 의존성(프로젝트가 사용하는 라이브러리) 목록을 관리합니다.

아래 명령어를 실행하여 `package.json` 파일을 생성합니다.

```bash
npm init -y
```

`-y` 플래그는 `npm init` 과정에서 묻는 모든 질문에 기본값으로 자동 응답해주는 옵션입니다. 이제 폴더 안에 `package.json` 파일이 생성된 것을 확인할 수 있습니다.

## 3단계: 기본 서버 파일 작성하기

이제 간단한 웹 서버를 만들어 보겠습니다. `index.js`라는 이름의 새 파일을 만들고, 아래 코드를 작성하세요. 이 코드는 Node.js에 내장된 `http` 모듈을 사용하여 "Hello, World!"를 응답하는 기본적인 서버를 생성합니다.

```javascript
// Node.js의 기본 http 모듈을 가져옵니다.
const http = require('http');

// 서버가 실행될 호스트 이름과 포트 번호를 정의합니다.
const hostname = '127.0.0.1';
const port = 3000;

// http.createServer 메서드를 사용하여 서버 인스턴스를 생성합니다.
// 이 서버는 요청(req)을 받을 때마다 응답(res)을 보냅니다.
const server = http.createServer((req, res) => {
  // 상태 코드 200 (OK)을 설정합니다.
  res.statusCode = 200;
  // 응답 헤더의 Content-Type을 text/plain으로 설정합니다.
  res.setHeader('Content-Type', 'text/plain');
  // "Hello, World!" 메시지로 응답을 종료합니다.
  res.end('Hello, World!\n');
});

// 서버가 지정된 포트와 호스트 이름에서 들어오는 요청을 수신 대기하도록 합니다.
server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});
```

## 4단계: 서버 실행하기

이제 작성한 서버를 실행할 차례입니다. 터미널에서 아래 명령어를 입력하세요.

```bash
node index.js
```

"Server running at http://127.0.0.1:3000/"이라는 메시지가 보이면 성공입니다. 이제 웹 브라우저를 열고 주소창에 `http://127.0.0.1:3000` 또는 `http://localhost:3000`을 입력하면 "Hello, World!"가 화면에 나타나는 것을 볼 수 있습니다.

서버를 중지하려면 터미널에서 `Ctrl + C`를 누르세요.

## (선택) 5단계: nodemon으로 개발 효율 높이기

개발 중에는 코드를 수정할 때마다 서버를 수동으로 껐다 켜야 하는 것이 번거롭습니다. `nodemon`은 파일 변경을 감지하여 자동으로 서버를 재시작해주는 매우 유용한 도구입니다.

아래 명령어로 `nodemon`을 개발 의존성(devDependencies)으로 설치합니다.

```bash
npm install nodemon --save-dev
```

설치가 완료되면 `package.json` 파일을 열고 `scripts` 부분을 아래와 같이 수정합니다.

```json
"scripts": {
  "test": "echo \"Error: no test specified\" && exit 1",
  "start": "node index.js",
  "dev": "nodemon index.js"
}
```

이제 아래 명령어를 사용하여 `nodemon`으로 서버를 시작할 수 있습니다.

```bash
npm run dev
```

이제 `index.js` 파일을 수정하고 저장할 때마다 서버가 자동으로 재시작되어 변경 사항이 즉시 반영됩니다.

## 마무리

축하합니다! 여러분은 방금 Node.js 프로젝트를 성공적으로 생성하고, 기본 웹 서버를 만들어 실행했습니다. `package.json`을 통해 프로젝트의 의존성을 관리하고, `npm`을 통해 다양한 패키지를 설치하며 프로젝트를 확장해 나갈 수 있습니다.

이제 여러분의 아이디어를 Node.js를 통해 현실로 만들어보세요!
